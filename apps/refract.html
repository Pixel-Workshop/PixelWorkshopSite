Refract

Refract is a laser puzzle game about precision, timing, and clean logic. Rotate mirrors, thread beams through narrow routes, and use prisms to split light into exactly the color paths each board demands.

Design

A puzzle game about clarity.

Refract is designed to feel readable from the first second. When a board gets harder, it should be because the logic is deeper, not because the interface is noisy.

The goal is short, satisfying puzzles that reward experimentation. Move a mirror, watch the beam change, and understand immediately why the new route works or fails.
