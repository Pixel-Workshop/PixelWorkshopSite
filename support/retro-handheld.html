FAQ

Frequently Asked Questions

You can add ROM files using iTunes File Sharing or the Files app. Connect your iPhone to your computer, open Finder (Mac) or iTunes (Windows), select your device, navigate to the Retro Handheld app under File Sharing, and drag your ROM files in. Alternatively, you can use the Files app on your iPhone to import ROMs directly.
Retro Handheld supports standard Game Boy (.gb) and Game Boy Color (.gbc) ROM files. Please note that you must only use ROMs for games you legally own.
To save your progress at any point, open the in-game menu and tap "Save State." You can have multiple save slots per game. To load a save state, open the menu and tap "Load State," then select the slot you want to restore.
Open the in-game settings and check the emulation speed setting. It should be set to 100% for normal speed. Adjusting it up or down lets you speed up or slow down gameplay intentionally, but 100% matches the original hardware speed.
Yes! Retro Handheld supports MFi (Made for iPhone) Bluetooth controllers. Simply pair your controller with your iPhone via Bluetooth settings, then open the app — it will automatically detect the controller.
In-game saves (using the game's own save system) are stored automatically. However, save states are separate — make sure to create a save state before closing if you want to resume mid-game. If in-game saves are still not persisting, try reinstalling the app (note: this will erase existing data).
Please check the App Store listing for current pricing information. Pricing may vary by region and is subject to change.
